# Overview
Module Name: 8pod Analytics by 8Pod

Module Type: Analytics Adapter

Maintainer: devs@8pod.com

# Description

Analytics adapter for Prebid provided by 8pod. It gets events from 8pod's ad unit and forwards supported events to Tealium.
Please, use it ONLY with eightpodBidAdapter.

# Analytics Adapter configuration example

```javascript
{
  provider: 'eightpod'
}
```
