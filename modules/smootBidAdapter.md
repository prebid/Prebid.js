# Overview

```
Module Name: Smoot Bidder Adapter
Module Type: Smoot Bidder Adapter
Maintainer: it.ops@smoot.ai
```

# Description

Connects to Smoot exchange for bids.
Smoot bid adapter supports Banner, Video (instream and outstream) and Native.
