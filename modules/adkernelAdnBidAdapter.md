# Overview

```
Module Name: AdKernel ADN Bidder Adapter
Module Type: Bidder Adapter
Maintainer: denis@adkernel.com
```

# Description

Connects to AdKernel Ad Delivery Network
Banner and video formats are supported.


# Test Parameters
```
    var adUnits = [
      {
        code: 'banner-ad-div',
        sizes: [[300, 250], [300, 200]],
```
