# Overview

```
Module Name:  Sublime Bid Adapter
Module Type:  Bidder Adapter
Maintainer: pbjs@sublimeskinz.com
```

# Description

Connects to Sublime for bids.
Sublime bid adapter supports Skinz and M-Skinz formats.
