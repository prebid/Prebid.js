# Logging & Troubleshooting
{:.no_toc}

To add Prebid Mobile logs for troubleshooting or to see successful bids simply import `PBLogging.h` and add the following code:

```objc
[PBLogManager setPBLogLevel:PBLogLevelAll];
```
