Testing/Debugging Guidance

  1. Make sure you have debug: true under pbjs.setConfig in this example code (be sure to remove it for production!)
  2. Make sure you have replaced <YOUR RESOURCE KEY> in this example code with the one you have obtained from the 51Degrees Configurator Tool
  3. Replace the placeholder tdlUrl in the example with your real TDL endpoint
  4. Pick a marketing preference in the PMP CMP overlay when it appears; the module reads localStorage['__51d_pmp_pref'] to derive the cloud's id.usage evidence
  5. Open DevTools Console in your browser and refresh the page
  6. Observe the enriched ORTB shown below: device, device.geo, and user.eids. Also check the console for [51Degrees RTD Submodule]: reqBidsConfigObj:
